# 📈 EGX Stock Analysis & Prediction System

This college project is an end-to-end stock analysis platform for the Egyptian Stock Exchange (CIB/COMI). It combines Data Engineering, Machine Learning, and real-time streaming to provide financial insights.

**Tech Stack:**
*   **Pipeline:** Python, Pandas, yfinance
*   **Machine Learning:** Scikit-Learn (Random Forest)
*   **Streaming:** Apache Kafka, Docker
*   **Frontend:** Streamlit, Plotly
